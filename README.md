## About me

I'm a backend engineer who spent 10 months building and shipping production software at **Solvei8**, where I worked on a cloud-edge Manufacturing Execution System running across **60+ global apparel factories**. I designed microservices, built RFID-based scanning systems, engineered Kafka event pipelines, and optimised MongoDB aggregation pipelines — all under real production constraints.

Before going full-time, I interned at **Viral Fission** in Mumbai, where I replaced a cron-based monitoring system with a Redis + Kafka real-time notification service and built a global exception handler that reduced mobile developer debugging time significantly.

I like building systems that handle complexity cleanly — distributed messaging, async pipelines, layered API design — and I care about writing code that's readable, testable, and documented.
